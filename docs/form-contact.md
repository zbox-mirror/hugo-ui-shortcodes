Форма для отправки сообщений по электронной почте автору сайта.

## Syntax

```
{{< form-contact subject="" token="" >}}
```

## ru.toml

```toml
[form.contact.name]
other = "Имя"

[form.contact.email]
other = "Email"

[form.contact.phone]
other = "Телефон"

[form.contact.message_placeholder]
other = "Введите сообщение..."

[form.contact.submit]
other = "Отправить"
```
